1. 프로젝트 개요

프로젝트 명

스마트 분리수거 가이드 앱 "EcoSort"

프로젝트 배경

  • 국내 분리수거 실천율 50% 미만 (환경부 통계)
  • 복잡한 분리수거 기준으로 인한 사용자 혼란
  • 지역별 상이한 분리수거 규정
  • 잘못된 분리수거로 인한 재활용률 저하 및 환경오염

비전 및 목표

비전

누구나 쉽게 올바른 분리수거를 실천할 수 있는 세상

목표

6개월 내 분리수거 정확도 30% 향상 및 사용자 인식 개선

2. 문제 정의 및 해결 방안

핵심 문제

  • 정보 부족: 복잡하고 지역별로 다른 분리수거 기준
  • 접근성 부족: 필요한 순간에 정보를 찾기 어려움
  • 동기 부족: 분리수거의 중요성과 효과에 대한 인식 부족
  • 습관 형성 어려움: 지속적인 실천을 위한 동기부여 부족

해결 방안

  • AI 기반 실시간 분류 가이드 제공
  • 게임화를 통한 재미있는 학습 경험
  • 지역 맞춤형 정보 제공
  • 커뮤니티 기반 인센티브 시스템

3. 대상 사용자 및 페르소나

주요 사용자 그룹

1차 타겟

20-40대 도시 거주자
(환경에 관심 있지만 실천에 어려움)

2차 타겟

학생 및 교육기관
(환경교육 목적)

3차 타겟

아파트 관리사무소 및 지자체

핵심 페르소나

김환경 (32세, 직장인)
  • 환경보호에 관심이 많지만 바쁜 일상으로 실천 어려움
  • 분리수거를 하려고 하지만 헷갈려서 포기하는 경우 많음
  • 스마트폰 활용도 높음
박학생 (22세, 대학생)
  • 환경에 대한 관심은 있지만 지식 부족
  • 게임과 소셜미디어를 자주 이용
  • 친구들과 경쟁하며 배우는 것을 선호

4. 핵심 기능 및 요구사항

MVP 핵심 기능

4.1 AI 분류 인식 기능

  • 카메라로 쓰레기 촬영 시 자동 분류 결과 제공
  • 95% 이상 정확도의 이미지 인식 AI
  • 음성 검색 기능 (예: "플라스틱 컵은 어디에?")

4.2 지역별 맞춤 정보

  • GPS 기반 현재 위치의 분리수거 규정 제공
  • 지자체별 수거일 및 배출 방법 안내
  • 대형폐기물 신청 연동 서비스

4.3 게임화 요소

  • 올바른 분리수거 시 포인트 적립
  • 레벨 시스템 및 배지 획득
  • 친구 초대 및 랭킹 시스템
  • 일일/주간 미션 제공

4.4 교육 콘텐츠

  • 분리수거 방법 동영상 가이드
  • 환경보호 효과 시각화
  • 퀴즈 및 미니게임

부가 기능 (추후 개발)

4.5 커뮤니티 기능

  • 지역별 분리수거 현황 공유
  • Q&A 및 팁 공유 게시판
  • 환경보호 챌린지 참여

4.6 인센티브 시스템

  • 포인트 교환 (커피쿠폰, 친환경 제품)
  • 지자체/기업 파트너십 혜택
  • 탄소포인트 연동

5. 사용자 스토리

US001: 사용자로서, 나는 쓰레기를 카메라로 찍어서 어떻게 분리수거해야 하는지 즉시 알고 싶다. 왜냐하면 매번 헷갈려서 잘못 버리는 경우가 많기 때문이다.
US002: 사용자로서, 나는 우리 동네의 분리수거 규정과 수거일을 확인하고 싶다. 왜냐하면 지역마다 규정이 달라서 이사 갈 때마다 혼란스럽기 때문이다.
US003: 사용자로서, 나는 분리수거를 재미있게 배우고 습관화하고 싶다. 왜냐하면 중요한 건 알지만 지속적으로 실천하기가 어렵기 때문이다.
US004: 사용자로서, 나는 내가 분리수거를 통해 환경에 미치는 긍정적 영향을 확인하고 싶다. 왜냐하면 작은 실천이 얼마나 의미있는지 알고 동기부여받고 싶기 때문이다.
US005: 사용자로서, 나는 분리수거를 잘하면 혜택을 받고 싶다. 왜냐하면 환경보호 실천에 대한 보상이 있으면 더 열심히 할 것 같기 때문이다.

6. 기술 요구사항

6.1 모바일 앱 (iOS/Android)

프론트엔드:

  • React Native 또는 Flutter
  • 카메라 API 연동
  • GPS 위치 서비스
  • 푸시 알림 시스템

백엔드:

  • Node.js + Express 또는 Python + FastAPI
  • PostgreSQL (사용자/콘텐츠 관리)
  • Redis (캐싱)
  • AWS S3 (이미지 저장)

6.2 AI/ML 시스템

이미지 인식:

  • TensorFlow/PyTorch 기반 CNN 모델
  • 쓰레기 분류 전용 학습 데이터셋 구축
  • Edge AI 최적화 (온디바이스 처리)

데이터 수집:

  • 국내 쓰레기 이미지 10만장 이상
  • 지역별 분리수거 규정 DB
  • 사용자 피드백 기반 모델 개선

6.3 외부 연동

  • 지자체 API (분리수거 정보)
  • 소셜 로그인 (카카오, 네이버, 구글)
  • 결제 시스템 (인앱 결제)
  • 푸시 알림 서비스
